# Sens简介


## Sens 定位

Sens是多合一AI的语言学习平台  
AI全家桶 All in One  
提供语言学习的一站式服务：  
ChatGPT、翻译、词典、点读卡、复读机、电子书、文本转语音(TTS)、发音评估，AI教练，语法分析，知识卡,单词卡  

无论是Web 还是App  
希望随时随地都能把时间用来学习
